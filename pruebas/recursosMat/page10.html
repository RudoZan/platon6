Objetivo del Juego

Completa todos los números objetivo usando los números básicos disponibles. Cada número objetivo debe completarse de dos maneras diferentes usando el mismo par de números básicos:

Ejemplo: Si seleccionas los números básicos 2 y 3:
• Puedes sumarlos: 2 + 3 = 5
• Puedes multiplicarlos: 2 × 3 = 6
• Ambos resultados (5 y 6) deben estar en la cuadrícula de números objetivo

Reglas Importantes

  • Cada par de números básicos se usa exactamente dos veces: una para suma y otra para multiplicación.
  • No puedes usar el mismo par más de dos veces.
  • Cada número objetivo debe completarse con ambas operaciones (suma y multiplicación) del mismo par.
  • Cuando completes un número objetivo, se marcará en verde.

Cómo Jugar

  1. Selecciona dos números básicos haciendo clic en ellos.
  2. Elige una operación (Sumar o Multiplicar).
  3. Si el resultado coincide con un número objetivo disponible, se marcará como completado.
  4. Repite el proceso hasta completar todos los números objetivo.

Consejos

  • Usa el botón "Pista" para resaltar un número objetivo que aún no has resuelto.
  • Piensa en pares de números que puedan generar resultados tanto por suma como por multiplicación.
  • Los números más grandes en la cuadrícula suelen requerir multiplicaciones con números básicos mayores.
