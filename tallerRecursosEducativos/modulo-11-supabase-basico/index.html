Módulo 11: Conectar con base de datos (Supabase)

Una base de datos permite que tu recurso guarde y muestre información real. Supabase hace este paso accesible para recursos que necesitan datos dinámicos.

Qué aprenderás

Al terminar este módulo serás capaz de:

  • Entender qué hace una base de datos en el sitio.
  • Configurar proyecto y tabla en Supabase.
  • Conectar el sitio para usar datos reales.
