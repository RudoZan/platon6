Módulo 04: Creación de sitio (HTML y JavaScript)

Este módulo te guía para armar la base de un sitio web educativo. No necesitas ser programador: verás cómo crear estructura, estilo e interactividad simple.

Qué aprenderás

Al terminar este módulo serás capaz de:

  • Construir la estructura básica de una página web.
  • Aplicar estilos claros y ordenados con CSS.
  • Agregar comportamiento dinámico con JavaScript.
  • Unir tus recursos sueltos en un solo sitio, y decidir cómo controlar quién puede acceder a él.
