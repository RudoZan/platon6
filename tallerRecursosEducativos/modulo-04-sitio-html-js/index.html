Módulo 04: Estructura web y Portal de Recursos

En este módulo comprenderás el rol de los archivos web (HTML, CSS y JavaScript) y cómo se relacionan entre sí. Además, aprenderás a estructurar un sitio web completo, integrando tus actividades en un Portal de Recursos que servirá como el producto tangible final de tu taller.

Qué aprenderás

Al terminar este módulo serás capaz de:

  • Comprender la función y relación mutua entre HTML (estructura), CSS (estilos) y JavaScript (comportamiento).
  • Organizar y planificar la estructura de carpetas de tu sitio para alojar múltiples recursos independientes.
  • Crear un Portal de Recursos unificado mediante una portada (index.html) que enlace a diferentes páginas y actividades del taller.
