Alternativa: Instalar el editor Cursor

Cursor es otro editor de código basado en Visual Studio Code con excelentes capacidades de inteligencia artificial. Si ya tienes una cuenta de pago o si tu institución financia esta herramienta, puedes utilizarla perfectamente para seguir todo este taller, ya que funciona de manera muy similar a Antigravity IDE.

Advertencia sobre costo

A diferencia de Antigravity IDE (que ofrece un acceso de vista previa completamente gratuito), Cursor es actualmente una herramienta de pago. Aunque solía contar con un plan gratuito funcional, en la actualidad requiere una suscripción mensual activa para poder realizar consultas y modificaciones asistidas por IA.

Pasos para instalar Cursor

  1. Entra a cursor.com desde tu navegador.
  2. Descarga el instalador para tu sistema operativo (Windows o Mac) presionando el botón de descarga principal.
  3. Ejecuta el archivo instalador en tu computador y sigue las instrucciones en pantalla.
  4. Inicia sesión con tu cuenta de Cursor o crea una cuenta nueva. Para habilitar las funciones de IA y edición automática, deberás configurar una suscripción de pago en tu panel de usuario de Cursor.

Si deseas evitar costes de suscripción o no cuentas con presupuesto institucional, te recomendamos volver a la página anterior y seguir el tutorial de instalación de Antigravity IDE.
