Módulo 02: IAs para programar (Elección e Instalación)

Aquí conocerás el panorama de las principales IAs orientadas a desarrollo de código disponibles hoy, aprenderás los criterios clave para elegir la herramienta que mejor se adapte a tu perfil docente y seguirás las instrucciones exactas para instalar y dejar listo tu asistente (como Antigravity IDE, Cursor o VS Code + Copilot).

Qué aprenderás

Al terminar este módulo serás capaz de:

  • Conocer las principales IAs para programar, a qué empresa pertenece cada una y sus características distintivas.
  • Evaluar y elegir la herramienta que más te convenga según tu equipo y preferencias de trabajo.
  • Instalar paso a paso tu IDE o asistente de IA para tener tu entorno de desarrollo 100% operativo y listo para crear.
