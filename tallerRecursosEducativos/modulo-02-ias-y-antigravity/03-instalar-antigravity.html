Instalar Antigravity IDE

En este taller usamos Antigravity IDE como nuestra herramienta de referencia. Es un editor de código completo diseñado desde cero alrededor de la inteligencia artificial, basado en Visual Studio Code. Esto significa que hereda toda la interfaz estándar del editor más popular del mundo, pero incluye su asistente de IA de forma nativa.

Versión gratuita y accesibilidad

Antigravity IDE se encuentra en su fase de vista previa pública (Public Preview) y su uso individual no tiene costo ni requiere tarjetas de crédito. Es una opción excelente para profesores de colegios y universidades que desean explorar la programación con asistencia inteligente sin incurrir en costos de licencias.

Instalar Antigravity IDE

  1. Entra a antigravity.google desde tu navegador.
  2. Descarga el instalador correspondiente a tu sistema operativo (Windows o Mac). La página principal te sugerirá de forma automática el archivo adecuado para tu computador.
  3. Abre el archivo descargado y sigue los pasos sencillos del asistente de instalación en tu equipo.
  4. Ejecuta el programa una vez completado el proceso. Verás una pantalla de bienvenida y una interfaz idéntica a Visual Studio Code, lista para usar.

Crear tu cuenta de GitHub (Recomendado)

Aunque Antigravity IDE tiene IA integrada de forma nativa y gratuita sin requerir configuración adicional, te recomendamos crear una cuenta gratuita en github.com en este momento. Más adelante en el Módulo 7 utilizaremos esta cuenta para publicar en internet tus recursos educativos interactivos de forma gratuita.

Si la página de descarga ha cambiado de diseño o tienes problemas de permisos para instalar programas en el computador de tu institución, coméntale la situación a cualquier chat de IA desde tu celular o navegador explicando qué error ves en pantalla para recibir sugerencias de solución rápida.
