Alternativa: Instalar VS Code y GitHub Copilot

Utilizar el editor estándar Visual Studio Code junto con la extensión de GitHub Copilot es una combinación sumamente robusta y popular en el ámbito profesional. Si prefieres usar esta opción, puedes seguir todo el taller de la misma manera, ya que la interfaz visual de edición es idéntica.

Acceso gratuito para docentes

Aunque GitHub Copilot es una herramienta de pago (con una suscripción mensual), es **completamente gratuita para profesores y estudiantes** de educación escolar y superior. Para acceder a este beneficio, debes registrarte en github.com y luego postular tu correo institucional en el programa GitHub Education.

Pasos para la instalación

  1. Instala Visual Studio Code: Descarga y ejecuta el instalador para tu sistema operativo (Windows o Mac) desde code.visualstudio.com.
  2. Crea tu cuenta de GitHub y solicita el beneficio académico: Regístrate en github.com y luego entra a education.github.com para postular como profesor (te pedirán verificar tu correo institucional o cargar una credencial de docente).
  3. Instala la extensión de Copilot: Abre VS Code, ve al panel de Extensiones en la barra lateral izquierda (ícono de bloques), busca "GitHub Copilot" y presiona "Install".
  4. Vincula tu cuenta: Haz clic en el ícono de perfil que aparecerá abajo a la izquierda en VS Code e inicia sesión con tu cuenta de GitHub autorizada para activar la IA.

Si la postulación académica en GitHub tarda unos días en ser aprobada y necesitas empezar el taller de inmediato sin realizar pagos, te sugerimos utilizar la instalación recomendada de Antigravity IDE descrita en las páginas anteriores.
