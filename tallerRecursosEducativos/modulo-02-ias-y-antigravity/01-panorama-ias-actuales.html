🔍 En revisión

Panorama de IAs actuales

Este taller no depende de una sola herramienta. Existen varias IAs orientadas específicamente a desarrollo de código, cada una hecha por una empresa distinta, y todas resuelven el mismo problema de fondo: ayudarte a escribir y modificar código a partir de instrucciones en lenguaje natural. Estas son las más relevantes hoy:

Antigravity IDE — Google

Un editor de código completo (basado en VS Code) diseñado desde cero alrededor del desarrollo asistido por inteligencia artificial. Cuenta con modo chat, autocompletado y un potente modo agente. Es la herramienta que usamos como referencia en este taller debido a que su versión de vista previa es gratuita e incluye todas las capacidades de IA integradas de forma nativa.

GitHub Copilot — GitHub (Microsoft)

La más veterana y masiva: nació sugiriendo líneas de código mientras escribías, y hoy también ofrece chat y modo agente. Se integra como extensión en VS Code y otros editores. Es de pago, aunque ofrece accesos gratuitos para estudiantes y docentes verificados.

Cursor — Anysphere

Otro editor de código completo basado en VS Code construido alrededor de la IA, con chat y modo agente. Originalmente contaba con una versión gratuita muy popular, pero actualmente sus funciones principales requieren una suscripción de pago.

Windsurf — Codeium

Otro editor construido alrededor de la IA, con un enfoque en flujos de trabajo guiados ("Cascade") donde la IA encadena varios pasos de forma más autónoma.

Gemini Code Assist — Google

La propuesta de Google, integrada en Android Studio, VS Code y sus herramientas de nube (Google Cloud).

Amazon Q Developer — Amazon (AWS)

Antes llamada CodeWhisperer. Pensada para integrarse bien con los servicios de Amazon Web Services, aunque también funciona como asistente de código general.

JetBrains AI Assistant — JetBrains

Integrada directamente en los editores de JetBrains (IntelliJ, PyCharm, WebStorm), útil si ya usas alguno de esos programas.

Esta lista va a seguir cambiando: aparecen herramientas nuevas todo el tiempo y las existentes se actualizan seguido. Lo importante no es memorizar esta lista, sino entender que todas comparten los mismos conceptos de fondo (modo chat, modo agente, prompts, memoria del proyecto) que vas a ver en las páginas siguientes de este módulo, aplicables a cualquiera que elijas.
