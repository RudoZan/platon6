Crear cuenta en GitHub

GitHub es donde vas a guardar el código de tu sitio y, más adelante en este módulo, donde vas a publicarlo para que cualquiera pueda verlo en internet. Crear una cuenta toma solo un par de minutos.

  1. Entra a github.com y haz clic en "Sign up" (Registrarse).
  2. Elige un nombre de usuario, tu correo y una contraseña. El nombre de usuario va a formar parte de la dirección de tu sitio más adelante, así que conviene que sea simple y profesional.
  3. Verifica tu correo electrónico cuando GitHub te lo pida.
  4. Elige el plan gratuito (Free): es más que suficiente para todo lo que vas a hacer en este taller.
Recuerda

Si en cualquier momento de este proceso no sabes qué hacer, puedes pedirle ayuda a la misma IA con la que ya estás trabajando:

  • Cuéntale qué quieres lograr y qué dudas tienes, puede guiarte paso a paso.
  • Si te aparece un mensaje de error que no entiendes, pégaselo y pídele que te lo explique.
  • Pídele que te sugiera cómo solucionarlo antes de intentar arreglarlo a ciegas por tu cuenta.
