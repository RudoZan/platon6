Módulo 08: Subir el sitio a GitHub

Este módulo te enseña a publicar tu recurso en la web. GitHub te da un lugar propio donde el sitio puede vivir y ser compartido fácilmente.

Qué aprenderás

Al terminar este módulo serás capaz de:

  • Crear tu repositorio en GitHub.
  • Subir los archivos del sitio paso a paso.
  • Publicar el proyecto con GitHub Pages.
  • Comprender el ciclo de desarrollo de software (diseño, pruebas y optimización).
